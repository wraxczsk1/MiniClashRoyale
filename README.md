# GitHub Pages Template

Tento projekt je připravený pro GitHub Pages.

## 📁 Struktura
- `index.html` – hlavní stránka
- `style.css` – styly
- `script.js` – JavaScript
- `/assets` – obrázky a další soubory

## 🚀 Deploy na GitHub Pages
1. Vytvoř repozitář
2. Nahraj soubory
3. Jdi do **Settings → Pages**
4. Vyber:
   - Branch: `main`
   - Folder: `/`
5. Ulož
